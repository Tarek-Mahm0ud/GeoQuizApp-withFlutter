## Getting Started

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK (>=3.0.0)
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Tarek-Mahm0ud/GeoQuizApp-withFlutter.git
```

2. Navigate to the project directory:
```bash
cd GeoQuizApp-withFlutter
```

3. Install dependencies:
```bash
flutter pub get
```

4. Run the app:
```bash
flutter run
```
